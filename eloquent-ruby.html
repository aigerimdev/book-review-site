Description

It's easy to write correct Ruby code, but to write great Ruby code, you need to think the Ruby way. This book helps you do that. It starts with simple examples and grows to advanced topics like metaprogramming and DSLs (Domain-Specific Languages). Russ Olsen writes in a friendly way and explains how Ruby works and why it works that way.

Reviews

Review #1: The first Ruby book I read in full within 24 hours

By: Peter Cooper

Date: February 26, 2011

This book is super easy to read but full of useful tips. The chapters are short and helpful. Russ writes in a fun way that makes you feel like a friend is explaining things to you. It helped me understand how to write clean, readable Ruby code.

Review #2: Eloquent Olsen

By: Fogus

Date: March 7, 2011

Even though I mostly use Java, I really enjoyed this Ruby book. Russ made the topic fun and interesting. I learned how Ruby can be beautiful and powerful, especially when it comes to creating small languages (DSLs) to solve problems. A great read for anyone new to Ruby!
